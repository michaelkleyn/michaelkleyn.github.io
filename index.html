Senior Project

For my senior project , I collaborated with Tao He and her lab at UCSF and SFSU, on a project funded by the National Science Foundation's LEAPS award.

The project aimed to explore various machine learning methods for predicting cell types using novel TCR data to expedite cancer diagnoses. During the project, I had the opportunity to train multiple statistical models using modern machine learning techniques. Using Python as my primary programming language, and leveraging the SKLearn API. By doing so, I helped the lab obtain performance metrics for their newly acquired dataset.

MemDex

MemDex is a chrome plugin that I'm developing to help augment my memory.

Memdex is a spaced repetition flashcard application which requires as little input from you as possible. All you need to do is select the bit of information you'd like to remember, click the "mem-it" button, and the large language model creates a flash card for you. The rest of the application is then just your standard spaced repetition flashcard application with shifting quiz windows, folders, and notifications.
